# JS Calculator

Aplikasi kalkulator sederhana berbasis web menggunakan HTML, CSS, dan JavaScript.

## Fitur
- Operasi dasar: tambah, kurang, kali, bagi
- Tombol hapus (⌫) dan clear (C)
- Support input keyboard
- Tampilan responsif

## Struktur File
js-calculator/
├── index.html
├── style.css
└── script.js

## Cara Menjalankan
1. Clone repository ini
2. Buka file `index.html` di browser
3. Kalkulator siap digunakan

## Teknologi
- HTML5
- CSS3
- JavaScript (Vanilla)
